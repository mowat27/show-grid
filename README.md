# show-grid

Creates a Swing window that displays a clojure data structure.  This is useful when working at the repl in development.

## Usage

See https://clojars.org/show-grid for installation

```clojure
(use 'show-grid.core)

(show-grid {:foo "bar" :bop "baz")
```
